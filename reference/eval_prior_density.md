# Evaluate the prior density at a point (internal)

Computes the analytical density of a `bvarnet_prior` at `x`.

## Usage

``` r
eval_prior_density(prior, x)
```

## Arguments

- prior:

  A `bvarnet_prior` object.

- x:

  Numeric scalar — the point at which to evaluate the density.

## Value

Numeric scalar — the density.
