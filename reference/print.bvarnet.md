# Print a bvarnet model object

Displays a brief summary of the fitted model: family, dimensions, Rhat,
divergences, chain return codes, priors, and total sampling time.

## Usage

``` r
# S3 method for class 'bvarnet'
print(x, ...)
```

## Arguments

- x:

  A `bvarnet` object.

- ...:

  Ignored.

## Value

`x` invisibly.
